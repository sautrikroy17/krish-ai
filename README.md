## 📖 Overview

**Krish AI** is a state-of-the-art, open-source AI bot designed to be your ultimate study partner, coding tutor, and conversational companion. 

Powered by **Meta's Llama 3.3 70B** running on **Groq's LPU Inference Engine**, Krish delivers responses at blisteringly fast speeds (750+ tokens per second). It features a custom-built, responsive Glassmorphism UI built with **Next.js** and **Tailwind CSS**.

*Built by a passionate Full-Stack Developer.*

---

## ✨ Core Features

- **⚡ Blazing Fast**: Powered by Groq for zero-lag conversational AI.
- **🎓 Smart Study Partner**: Get personalized study plans, analogies, and detailed explanations for complex topics.
- **💻 Coding Tutor**: Debug issues, brainstorm architectures, and learn modern frameworks.
- **💜 Emotionally Intelligent**: Features a custom system prompt designed to provide empathetic and genuine conversation.
- **🎨 Glassmorphism UI**: A stunning, animated user interface with dark mode and streaming responses.
- **🔒 Bring Your Own Key**: Runs securely using your own Groq API key with no data surveillance.

## 🚀 Getting Started

To run this project locally, follow these steps:

### 1. Clone the repository
```bash
git clone https://github.com/sautrikroy17/krish-ai.git
cd krish-ai
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Set up Environment Variables
Create a `.env.local` file in the root directory and add your Groq API key:
```env
GROQ_API_KEY=gsk_your_api_key_here
```

### 4. Run the Development Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
